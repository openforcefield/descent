# Release History

## Development Version

Current development version. Not yet released.

### New Features

- Initial implementation of differentiable force field parameter optimization
- Support for energy, thermodynamic, and dimer targets
- PyTorch-based optimization with automatic differentiation
- Integration with OpenFF Toolkit for force field manipulation
